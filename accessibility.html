無障礙說明與網站導覽

本網站依據國家通訊傳播主管機關公布之「網站無障礙規範」設計與製作， 致力達到 AA 等級，讓所有使用者都能無障礙地取得資訊。

快速鍵（Accesskey）

本網站設定下列快速鍵，方便您快速到達主要區塊：

網站快速鍵對照表
快速鍵功能
Alt + U上方導覽連結區
Alt + C中央主要內容區塊
Alt + L左方區塊選單（具左側選單的內頁適用）
Alt + B下方頁尾資訊區

快速鍵啟用方式因瀏覽器而異：Windows 多為 Alt 加字母； 部分瀏覽器需 Alt+Shift 加字母；macOS 多為 Ctrl+Option 加字母。

鍵盤操作

  • 使用 Tab 鍵可依序移動到網頁中的連結與表單欄位，Shift+Tab 可往回移動。
  • 移動到連結或按鈕時按 Enter 即可開啟；核取方塊等可用 空白鍵 切換。
  • 所有操作皆可在不使用滑鼠的情況下完成，且目前焦點位置會有明顯外框標示。

調整文字大小

本網站文字採相對單位設計，您可使用瀏覽器內建功能放大或縮小： 按 Ctrl++ 放大、Ctrl+- 縮小、Ctrl+0 還原（macOS 改用 ⌘）。 放大至 200% 仍可正常閱讀，不會破版。
